Pine is a static site generator. It uses Branches at its core to convert a directory of content to a static collection of HTML and CSS. You can download the latest .phar executable here.

After that, you can add the executable to the $PATH and just use pine. Finally, you can create a new site with:

pine new blog # Where blog is the name of your site
